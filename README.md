# Three.js 魔方

Three.js + Typescript + Webpack 开发的魔方项目

|          |            |
| -------- | ---------- |
| 编译环境 | node       |
| 运行环境 | 浏览器     |
| language | typescript |
| 打包工具 | webpack    |
| 渲染     | Three.js   |

## Usage

### 安装

```shell
    npm i
```

### 运行

```shell
    npm run dev
```

### 打包

```shell
    npm run build
```
